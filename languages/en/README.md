# eMathica

> Math creation, beyond just calculation.

eMathica is a math creation platform that enables everyone to express, explore, and create with mathematics.

This project is currently documented primarily in Chinese. The English version is a work in progress.

## Quick Navigation

| Repository | Description |
|------------|-------------|
| [eMathica Core](https://github.com/nightcreek/eMathica-Core) | Main app — Math creation platform |
| [OpenMathInk Collector](https://github.com/nightcreek/OpenMathInk-Collector) | Handwriting data collection tool |
| [Shared Libraries](https://github.com/nightcreek/SharedLibraries) | Shared Swift Packages |
| [OpenMathInk Dataset](https://github.com/nightcreek/MathInk-Dataset) | Open-source math handwriting dataset (planned) |
