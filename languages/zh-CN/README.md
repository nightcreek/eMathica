# eMathica

> 数学创作，不止于计算。

eMathica 是一个数学创作平台，让每个人都能用数学表达、探索和创作。

## 目录

1. [Philosophy (哲学)](./Philosophy/) — 为什么存在
2. [Current Reality (当前状态)](./CurrentReality/) — 当前实际状态
3. [Future Possibilities (未来可能)](./FuturePossibilities/) — 未来方向（非承诺）
4. [Repository Index (仓库索引)](./RepositoryIndex/) — 仓库概览
5. [Policies (知识标准)](./Policies/KnowledgeStandard.md) — 文档管理标准与规范
